## üõ†Ô∏è How It Works (The Core Loop)

RiskWise isn't just a chatbot. It is a **Deterministic Orchestration Engine**.

1.  **Input Vector**: User defines Product, Origin, Destination, and Mode (e.g., "Semiconductors, Taiwan -> Germany, Air").
2.  **Agent Dispatch**: The `RiskWiseOrchestrator` spins up three dedicated experts:
    *   `PoliticalAgent`: Scans for regional instability, sanctions, and conflict.
    *   `LogisticsAgent`: Evaluates port congestion, weather patterns, and route safety.
    *   `TradeAgent`: Checks for tariffs, compliance roadblocks, and regulatory shifts.
3.  **Synthesis**: The `ReportingAgent` aggregates these conflicting signals into a single "Risk Score" (0/20) and a cohesive narrative.
4.  **Visualization**: Results are rendered on a 3D Pydeck globe with live metrics.
